## CI/CD Integration

This project is integrated with GitHub Actions CI.

- Playwright tests run automatically on every push to the `main` branch
- CI uses headless Chromium for stability
- UI tests are hardened to handle dynamic content such as ads
- Both API and E2E tests are validated before merge
